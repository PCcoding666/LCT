## 🇨🇳 中文版本

### 什么是 LCT？
LCT (LiveCaptions Translator) 是一款 macOS 实时字幕翻译工具。它可以捕获音频、识别语音并实时翻译成目标语言。

### 主要功能
- **实时语音识别**：支持 14 种语言
- **AI 智能翻译**：支持翻译成 10 种语言
- **悬浮字幕窗口**：可拖动的半透明字幕
- **历史记录**：自动保存所有翻译内容

### 快速开始
1. **下载安装**：从 Releases 下载 .dmg 文件
2. **首次设置**：授予权限、安装 Ollama、下载 AI 模型
3. **开始使用**：点击 Start 按钮开始翻译

### 基本操作
- 开始/停止：`⌘ + Space`
- 打开设置：`⌘ + ,`
- 显示/隐藏悬浮窗口：`⌘ + O`
- 复制翻译：`⌘ + ⇧ + C`

### 常用设置
- **源语言**：设置 > Speech Recognition > Source Language
- **目标语言**：设置 > Translation > Target Language
- **音频捕获**：设置 > Audio Capture > 选择系统音频或麦克风

## 🇺🇸 English Version

### What is LCT?
LCT (LiveCaptions Translator) is a real-time caption translation tool for macOS. It captures audio, recognizes speech, and translates it to your target language in real-time.

### Main Features
- **Real-time Speech Recognition**: Supports 14 languages
- **AI Translation**: Translates to 10 languages
- **Floating Caption Window**: Draggable, semi-transparent captions
- **History**: Automatically saves all translations

### Quick Start
1. **Download & Install**: Download the .dmg file from Releases
2. **First-time Setup**: Grant permissions, install Ollama, download AI model
3. **Start Using**: Click the Start button to begin translation

### Basic Operations
- Start/Stop: `⌘ + Space`
- Open Settings: `⌘ + ,`
- Toggle Overlay: `⌘ + O`
- Copy Translation: `⌘ + ⇧ + C`

### Common Settings
- **Source Language**: Settings > Speech Recognition > Source Language
- **Target Language**: Settings > Translation > Target Language
- **Audio Capture**: Settings > Audio Capture > Choose system audio or microphone

## 常见问题 / FAQ

### 🇨🇳 中文

**Q: 为什么没有声音被识别？**  
A: 检查是否点击了 Start 按钮、音频权限是否授予、源语言是否选择正确。

**Q: 翻译速度慢怎么办？**  
A: 尝试使用更小的 AI 模型（如 gemma2:2b），或降低上下文条目数量。

**Q: 如何授予权限？**  
A: 系统设置 > 隐私与安全性 > 分别授予麦克风、屏幕录制、语音识别权限。

**Q: Ollama 是什么？**  
A: Ollama 是本地 AI 引擎，用于运行翻译模型。首次设置时会自动安装。

### 🇺🇸 English

**Q: Why is no sound being recognized?**  
A: Check if you clicked the Start button, audio permissions are granted, and the source language is correctly selected.

**Q: Translation is slow, what should I do?**  
A: Try using a smaller AI model (like gemma2:2b), or reduce the number of context entries.

**Q: How do I grant permissions?**  
A: System Settings > Privacy & Security > Grant microphone, screen recording, and speech recognition permissions separately.

**Q: What is Ollama?**  
A: Ollama is a local AI engine used to run translation models. It will be automatically installed during first-time setup.
