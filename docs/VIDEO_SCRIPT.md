# 🎬 LiveCaptions Translator (LCTMac) 项目介绍视频脚本

**视频时长**：约 3 分钟
**视频风格**：极客风、快节奏、B-Roll展示与屏幕录制结合
**背景音乐**：节奏轻快、科技感的电子节拍器（Synthwave 或 Lofi Chill）

---

### [00:00 - 00:30] 第一部分：痛点引入与破局

**【画面】**
- （B-Roll）一个用户戴着耳机，正在看一段无字幕的外语技术讲座视频，或者在开跨国 Zoom 视频会议。眉头紧锁，显然听得有些吃力。
- 突然，镜头推近屏幕边缘。一个拥有高级“毛玻璃”质感的窄边框透明悬浮窗优雅地滑入画面。
- 视频里的英文原声一出，悬浮窗上立刻以肉眼可见的极快速度，流式吐出了“经过精美排版和口语去噪”的中文翻译。

**【旁白（Voiceover）】**
> “跨国会议、无字幕外网生肉、外语连麦……语言障碍总是打破信息获取的流畅感。传统的语音翻译软件，要么网络延迟让人抓狂，要么把‘嗯、啊、那个’全翻出来，词不达意，并且那巨大厚重的窗口总是遮挡你的屏幕。”
>
> “今天，为你介绍专为 macOS 打造的原生实时翻译神器：**LiveCaptions Translator (LCTMac)**。极致轻量、零延迟感知、而且——完全本地化。”

### [00:30 - 01:20] 第二部分：核心卖点展示 (UI与原生性能)

**【画面】**
- （屏幕录制）展示 LCTMac 的“渐进式毛玻璃 HUD”。鼠标不在窗口时，它就像融化在桌面壁纸里，只有滚动的翻译字幕浮现在空中。
- 鼠标移入（Hover），顶部的设置栏和底部的音频跳动条顺滑地淡入展现。
- 开启 Activity Monitor (活动监视器)，特写 LCTMac 极低的内存（十几 MB）与 CPU 占用。

**【旁白】**
> “LCTMac 摒弃了臃肿的跨平台框架。纯净的 Swift 与 SwiftUI 铸就了它无以伦比的性能与与生俱来的 macOS 基因。它引入了极简的毛玻璃悬浮交互设计。当你专注工作时，它是一条透明的滚动画框，绝不喧宾夺主；当鼠标悬停，控制面板如魔法般浮现，掌控一切。”
>
> “它不仅长得好看，更轻得不可思议。基于苹果原生的 SFSpeechRecognizer 框架捕获系统级音频与语音识别，几乎零负担。”

### [01:20 - 02:10] 第三部分：真正的“魔法”——单路流式翻译管道与大模型

**【画面】**
- （架构演示动画 / 示意图）麦克风图标 -> 生成了一段杂乱带有“uh, you know”的英文草稿。
- 草稿进入一个发光的方块（Ollama 图标）。
- 输出分为两层文本：原句被浓缩成了小字注脚，而精炼的完美中文用大字加粗流式打在屏幕上。
- （屏幕录制）打开设置面板，展示填写了 `qwen3.5:4b-mlx` 模型的过程。

**【旁白】**
> “但 LCTMac 最迷人的魔法，在它的‘大脑’里。我们重构了字幕的流式管道处理引擎（Caption Segmenter）。对于随口而出的破碎语音，它并非死板地直翻。”
>
> “我们无缝对接了 Ollama 本地大语言模型。只需要部署一个像 Qwen 4B 这样的甜点级本地模型，LCTMac 就能在后台执行高难度的‘单次提问双向输出’指令：它会一秒去除你听到的口头禅、纠正语病，再流式输出最高质量的本土化翻译。”
>
> “而且别忘了，这一切都在你的 Mac 本地沙盒里进行。**你的会议机密，一个字节都不会传到云端。**”
