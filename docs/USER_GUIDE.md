# 用户使用手册：LCTMac 实时语音翻译助手 🎧

欢迎使用 LCTMac！这是一款能够“听到”并“翻译”你的视频会议、播客、视频的实时翻译悬浮窗工具。

## 📥 准备工作与初次启动

1. **安装大模型底层 (Ollama)**
   - 本应用基于本地大语言模型进行翻译，因此请先前往 [Ollama 官网](https://ollama.com) 下载并安装 Ollama。
   - 打开终端，输入命令下载推荐翻译模型：
     ```bash
     ollama run qwen3.5:4b-mlx
     ```
     等待下载完成后，保持 Ollama 在后台运行。

2. **启动 LCTMac**
   - 首次启动 LCTMac 时，系统可能会请求 **麦克风权限** 与 **语音识别权限**。请点击“允许”，以便 LCT 能捕获声音并将其转写为文字。

---

## 🎮 界面与交互指南

LCTMac 采用了全新的 **“极简毛玻璃悬浮条” (Glassmorphism HUD)** 交互模式，它就像一条精致的翻译字幕带，不阻挡任何有效内容。

### 1. 悬停展开控制栏 (渐进式交互)
- 默认情况下，LCTMac 只有一块半透明的毛玻璃区域。它默默滚动显示翻译字幕，不会打扰你的视线。
- 当你需要进行设置或者开关录制时，**将鼠标移动（悬停）进入窗口上方**，你会看到顶部的功能栏和底部的控制栏平滑地展开。
- 移开鼠标后，它们会自动隐身。

### 2. 核心操作介绍
当鼠标悬停并展开面板后：
- **Start / Stop (底部绿色/红色按钮)**：点击开始捕获音频并进行识别翻译，再次点击停止。
- **音量指示条 (底部绿条)**：实时反馈它正在听到的声音大小。
- **设置按钮 (右上角 ⚙️)**：打开配置中心。你可以在这里修改大语言模型、更改要翻译的源语言（支持英、日、中等 14 种语言）和目标语言。
- **历史记录 (右上角 🕒)**：查看并复制之前所有的翻译原文和翻译文稿。
- **Scroll to Bottom**：如果不小心向上滚动了历史记录，翻译自动追踪会暂停。点击浮现的“⬇️ Scroll to Bottom”按钮可以一键回到底部并恢复自动滚屏跟进。

## ⚙️ 核心设置（推荐配置）

进入设置页面 (⚙️) 后，请确认以下高级选项，以获取最佳体验：

- **Ollama Model**：推荐输入 `qwen3.5:4b-mlx`。这是一个 40 亿参数量的极速模型，能在保证翻译和润色（去除“啊、嗯”等口语）质量的同时，极大地降低延迟。
- **Context Aware (上下文感知)**：建议开启。开启后，遇到难以拆分的长难句时，AI 会结合你上一句说过的话综合翻译，不会显得断章取义。

## 📝 字幕排版说明

为了你能在繁忙的工作或者视频中快速捕获重点，我们在字幕卡片上做了清晰的层级区分：
- **较小、浅灰色的字体**：这是原音识别下来的口语草稿（带有语气词，供你参考）。
- **较大、加粗的彩色字体**：这是经过大模型提炼、润色并翻译出来的最终完美版文本。如果文本呈现青色（Cyan），代表模型“正在极速思考并生成中”；呈现主色（黑色/白色，视系统外观而定）则代表生成已完成并固化。

准备好了吗？点击底部的 **Start** 按钮，开始你的无语言障碍交流之旅吧！
